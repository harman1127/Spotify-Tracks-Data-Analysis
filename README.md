# 🎧 Spotify Insights: Music Data Analysis & Statistical Study

---

## 1. Overview  
Spotify Insights is an end-to-end data analysis project that explores music data to uncover patterns in song popularity, audio features, and listening behavior.

The project focuses on extracting meaningful insights using Exploratory Data Analysis (EDA) and statistical techniques to understand what factors contribute to a song’s success.

---

## 2. Objectives  

- Analyze patterns in song popularity across genres  
- Identify key audio features influencing song success  
- Study listening behavior based on track duration  
- Perform statistical validation on dataset  
- Understand relationships between features and popularity  

---

## 3. Features  

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA) with meaningful visualizations  
- Statistical testing (T-test, ANOVA, Normality Test)  
- Correlation analysis of audio features  
- Insight generation from real-world music data  
- Professional visualizations using Seaborn  

---

## 4. Tech Stack  

- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- SciPy (Statistical Analysis)  
- VS Code  

---

## 5. Results  

- Identified genres that frequently produce high-popularity songs  
- Observed that most songs fall within 2–4 minute duration  
- Found that energy levels influence the likelihood of a song becoming popular  
- Determined that popularity is concentrated among a small number of high-performing tracks  
- Statistical tests revealed relationships between features and song success  

---

## 6. Workflow  

### 🔹 Step 1: Data Cleaning & Preparation  

- Load dataset  
- Remove duplicates and unnecessary columns  
- Handle missing values  
- Prepare dataset for analysis  

---

### 🔹 Step 2: Exploratory Data Analysis (EDA)  

- Analyze high-popularity genres  
- Study danceability distribution across genres  
- Analyze energy impact on song success  
- Examine duration patterns  
- Visualize popularity distribution  

---

### 🔹 Step 3: Statistical Analysis  

- Perform correlation analysis  
- Visualize relationships using heatmap  
- Conduct normality testing (Shapiro Test)  
- Perform hypothesis testing (T-test, ANOVA)  
- Analyze statistical significance of features
